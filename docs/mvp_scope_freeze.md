# MVP Scope Freeze

“A deterministic earnings-call guidance-change assistant intended to help a retail user review a transcript faster and more consistently than unaided manual review.”

## In Scope
- Deterministic guidance-change labeling with a closed label set.
- Evidence-backed extraction from transcript text.
- Human-evaluable outputs for agreement and speed benchmarking.

## Out of Scope
- LLM-based interpretation or narrative generation.
- Live API integrations or real-time trading workflows.
- Broad multi-signal expansion beyond guidance-change MVP validation.

## Deprioritized (Not Deleted)
Tone, friction, risk language, IBKR integration, and UI are not deleted, only deprioritized for the 17-day MVP.

## Primary and Secondary Metrics
Primary metric is guidance-change label agreement, with completion time as secondary metric.
